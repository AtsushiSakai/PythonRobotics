Getting Started
===============

Requirements
-------------

-  Python 3.6.x
-  numpy
-  scipy
-  matplotlib
-  pandas
-  `cvxpy`_

.. _cvxpy: http://www.cvxpy.org/en/latest/


How to use

   conda command.
