Normal vector calculation of a 3D triangle
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

A 3D point is as a vector:

.. math:: p = [x, y, z]

When there are 3 points in 3D space, p1, p2, p3,

we can calculate a normal vector n of a 3D triangle which is consisted of p1, p2, p3.
